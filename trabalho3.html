Qual é a diferença entre linguagem de programação Server Side e Client Side? Cite exemplos de cada tipo.

Server Side: O código é executado no servidor. Ele processa solicitações, acessa bancos de dados e gera conteúdo dinâmico que é enviado ao cliente.

Exemplos:

  • PHP: Usado amplamente em desenvolvimento web;
  • Python (Django, Flask): Frameworks populares para desenvolvimento de aplicações web;
  • Ruby (Ruby on Rails): Usado para construir aplicações web rápidas e eficientes;
  • Java (Spring): Usado em aplicações corporativas e web.

Client Side: O código é executado no navegador do usuário. Ele manipula a interface do usuário e interações em tempo real.

Exemplos:

  • JavaScript: A principal linguagem para programação do lado do cliente, permitindo a manipulação do DOM;
  • HTML/CSS: Usados para estruturar e estilizar páginas web;
  • TypeScript: Um superconjunto do JavaScript, que adiciona tipagem estática;
  • Frameworks JS: Como React, Angular e Vue.js, que facilitam a construção de interfaces ricas.
